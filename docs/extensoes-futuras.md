# Extensões futuras

Ideias para evolução do projeto:

- Orquestração multi-tenant para hospedar vários webtops isolados.
- Integração com autenticação centralizada (OAuth2, SSO institucional).
- Pipelines de build automatizados para imagens customizadas e testes de regressão.
- Integração com CI/CD para deploy e atualizações seguras.
